Class: WindowManager

WindowManager(type)

new WindowManager(type)

Message utils takes care of messaging between multiple windows. Messages can be transfered direcly or forwarded between windows. Is responsible for keeping track of all sessions, handles message reception, message forwarding and message sending. When a window is open it cannot be acessed we need to wait for the ready message.
